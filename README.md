# Build a Well for Charity: Water

This is a one-page educational browser game for college students, built with HTML, CSS, and JavaScript.

## Game Concept

You play as a student volunteer helping raise funds and plan a clean water well project.
Your mission has 3 stages:

1. Fundraising
2. Choosing a well site
3. Building a sustainable water well

Each decision changes your score and teaches a short lesson about clean water access, sustainability, and smart planning.

## Features

- Start screen, gameplay screen, and ending screen
- Score and stage progress display
- Progress bar for the 3-stage mission
- Decision-based rounds with educational feedback after each choice
- Final outcome based on total score

## How to Run

1. Open `index.html` in your browser.
2. Click **Start Mission**.
3. Select one choice per stage and read the feedback.
4. View your final result and use **Play Again** to try a better strategy.
